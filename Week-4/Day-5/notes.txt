What is Keras?

    Keras is a high-level Deep Learning API used to build and train neural networks easily.

It works on top of:

    TensorFlow

Why Keras is Popular?
    Easy to use
    Less code
    Fast development
    Used in real-world AI projects

CPU             vs          GPU
    CPU	                        GPU
    Slower	                    Faster
    Few cores	                Thousands of cores
    Good for small tasks	    Good for Deep Learning
    Sequential processing	    Parallel processing

Why GPU is Important in Deep Learning?

Deep learning involves:

    Large matrix operations
    Millions of calculations

GPU performs:

    Parallel computation
    Faster training
