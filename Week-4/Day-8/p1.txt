1. What is RNN?
Definition

An RNN is a neural network where:

output from previous step
is fed back into the network.

This creates a memory mechanism.

Simple Idea

Traditional Neural Network:

Input → Output

RNN:

Input → Hidden State → Output
          ↑
     Previous Memory



Real World Analogy

Imagine reading a sentence:

"I went to the bank to deposit money."

To understand:

"bank"

you need previous words:

deposit
money

RNN works similarly:

it remembers previous context.

3. RNN Architecture
Basic RNN structure:
x₁ → h₁ → y₁      ↓x₂ → h₂ → y₂      ↓x₃ → h₃ → y₃
Where:


x = input


h = hidden state (memory)


y = output



Visualization
Input Word     ↓┌─────────┐│ Hidden  │ ← Previous Memory│ State   │└─────────┘     ↓ Output

4. Hidden State in RNN
The hidden state stores:


previous information


context


memory


Formula:
hₜ = f(hₜ₋₁, xₜ)
Meaning:


current hidden state depends on:


previous hidden state


current input

8. RNN Memory Mechanism
RNN has:
Short-Term Memory
It remembers:


recent information


But struggles with:


very long sequences.



Example Problem
Sentence:
"The food cooked yesterday by my grandmother was delicious."
To understand:


"delicious"


model needs long context.
Basic RNN struggles here.

9. Vanishing Gradient Problem
This is the biggest issue in RNN.
During backpropagation:


gradients become extremely small.


As a result:


earlier information gets forgotten.



Visualization
Long Sequence ↓Gradient becomes smaller ↓Memory fades

10. Exploding Gradient Problem
Sometimes gradients become:


extremely large


This causes:


unstable training.



Solutions


Gradient clipping


Better architectures



11. LSTM (Long Short-Term Memory)
LSTM is an improved RNN.
Designed to:


remember long-term dependencies.



Why LSTM?
Basic RNN:


forgets old information.


LSTM:


selectively remembers important information.



LSTM Architecture
Contains:


Forget Gate


Input Gate


Output Gate



LSTM Visualization
Input  ↓Forget Gate  ↓Input Gate  ↓Memory Cell  ↓Output Gate

Real World Analogy
Imagine studying for exams.
You:


forget unimportant details


remember important formulas


LSTM works similarly.
